- `gg` - go to start of file.
- `:$` - go to end of file.
- `viw` - select entire word.
- `ciw` - replace entire word.
- `0` - go to start of line.
- `$` - go to end of line.
- `Y` - copy entire line.
- `V` - select entire line.
- `dd` - delete entire line. (`3dd` - delete 3 entire lines)
- `r` - replace character.
- `x` - delete character.
